# mimiccss

> A mimic of brazilian css preprocessor
install your CLI locally:

```sh
$ [sudo] npm link
```

Run:

```sh
$ mimiccss
```

```sh
$ mimiccss -x
```

```sh
$ mimiccss -y
```

Run unit tests with mocha:

```sh
$ npm test
```

Publish:

```sh
$ npm punlish
```
